# XRAY-config

VPN configurations for Russia

## Subscriptions
- Blacklists: VLESS, Shadowsocks, Hysteria2, Trojan, VMess
- Whitelists: CIDR, SNI
- Tor Bridges

## Quick Start
1. Download client (v2rayN, Karing, Streisand, etc)
2. Add subscription URL
3. Update and test latency
4. Connect

## Mirrors
Use Yandex Translate, jsDelivr, Statically, Githack if GitHub blocked
